What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that provides a structured representation of HTML and XML documents, allowing programs and scripts to dynamically access and manipulate the content, structure, and style of a document. It treats documents as a tree structure where each node represents a part of the document, such as elements, attributes, and text.

How do you select an element from the DOM?

To select an element from the DOM (Document Object Model) in JavaScript, you can use various methods

01

getElementById(): Selects an element by its unique ID.

Example: document.getElementById('myElement')

02

getElementsByClassName(): Selects all elements with a specific class name (returns a collection).

Example: document.getElementsByClassName('myClass')

03

querySelector(): Selects the first element that matches a CSS selector.

Example: document.querySelector('.myClass')

04

querySelectorAll(): Selects all elements that match a CSS selector (returns a NodeList)

Example: document.querySelectorAll('div.myClass')

What is event delegation in the context of the DOM, and why is it useful?

Event Delegation in the DOM refers to a technique where you attach a single event listener to a parent element to handle events on its child elements. Instead of adding separate listeners to each child, the parent element’s listener uses the event’s target property to identify which child triggered the event.

Example 

                    
document.getElementById('parent').addEventListener('click', (event) => {
if (event.target.matches('.child')) {
console.log(`Child clicked: ${event.target.textContent}`);
 }
});

How do you manipulate an element's attributes and styles using the DOM?

The DOM (Document Object Model) allows you to dynamically modify an element’s attributes and styles, enabling interactive and responsive web designs. Here's how you can do it:

Manipulating Attributes

Get an Attribute: Use getAttribute().

                
const link = document.querySelector('a');
console.log(link.getAttribute('href')); // Outputs the href value

Set an Attribute: Use setAttribute().

                
const link = document.querySelector('a');
console.log(link.getAttribute('href')); // Outputs the href value

Remove an Attribute: Use removeAttribute().

                
link.removeAttribute('target'); // Removes the target attribute
